    Enable Your Vibe

    Vibe Coding 핸즈온 세션


    AI-Powered Development Hands-on — 반나절, 3–4시간

    10분 | 세션 소개

    안녕하세요! Enable Your Vibe 핸즈온 세션에 오신 것을 환영합니다. 오늘은 약 3-4시간에 걸쳐 Vibe Coding을 직접 경험하는 시간을 갖겠습니다. 먼저 세션 전체 소개와 로지스틱을 안내드리고, 바로 본 내용으로 들어가겠습니다.

    오늘의 목표

    1. AI 기초 이해 — AI Fluency, Vibe Coding 개념과 Claude AI의 핵심 특징 파악
    2. 핵심 도구 습득 — Claude Code, MCP, Skills 등 개발 도구 직접 사용
    3. 실전 구현 경험 — Genie MCP 서버부터 웹 UI까지 전체 파이프라인 구축
    4. 배포까지 완성 — Databricks Apps로 실제 서비스 배포 경험
    오늘 세션의 네 가지 목표입니다. 첫째, AI와 협업하기 위한 기초 개념을 이해합니다. 둘째, Claude Code, MCP, Skills 등 실제 도구를 직접 사용해봅니다. 셋째, 자연어로 데이터를 조회하는 MCP 서버부터 웹 UI까지 전체를 구현합니다. 넷째, 만든 것을 Databricks에 배포하여 실제 서비스로 만듭니다. 기초부터 배포까지 전체 흐름을 하루에 경험하는 것이 핵심입니다.

    무엇을 만드나요?

    자연어로 데이터를 조회하는 AI 에이전트 웹 앱을 만들고 배포합니다

    자연어 질문 "이번 달 매출은?"
    →
    Genie MCP SQL 생성 & 실행
    →
    Web UI 결과 시각화
    →
    Databricks 서비스 배포

    이 모든 것을 Vibe Coding으로 — 코드를 직접 작성하지 않고, AI에게 의도를 전달하여 구현합니다

    오늘 만들 것의 전체 그림입니다. 사용자가 자연어로 데이터 관련 질문을 하면, Genie MCP 서버가 SQL을 생성하고 실행해서 데이터를 가져오고, 웹 UI에서 결과를 보여주는 앱을 만듭니다. 그리고 이것을 Databricks에 배포하여 실제로 접근 가능한 서비스로 만듭니다. 중요한 점은 이 모든 과정을 Vibe Coding으로 진행한다는 것입니다.

    환경 설정

    아래 3단계로 실습 환경을 준비합니다 

    # 1. 프로젝트 클론
git clone https://github.com/Aiden-Jeon/enable-your-vibe-exercise
cd enable-your-vibe-exercise
    # 2. 패키지 설치 (모든 실습 의존성 한 번에)
uv sync
    # 3. 설치 확인
uv run python -c "import fastmcp; print('Ready!')"

    uv sync 한 번으로 모든 실습 패키지가 설치됩니다 — 섹션별 pip install은 필요 없습니다!

    도구 확인을 마쳤으면 이제 실습 환경을 설정합니다. 먼저 프로젝트를 git clone으로 받습니다. 그 다음 프로젝트 루트에서 uv sync를 실행하면 pyproject.toml에 정의된 모든 의존성이 한 번에 설치됩니다. fastmcp, httpx, fastapi, uvicorn 등 실습에 필요한 패키지가 모두 포함되어 있습니다. 마지막으로 import 테스트로 설치가 잘 되었는지 확인합니다. 이후 각 섹션에서 별도로 pip install을 할 필요가 없습니다.

    세션 진행 방식

    강의 (Lecture)

    핵심 개념 설명 — 슬라이드 중심, 배경 지식과 이론을 전달합니다

    데모 (Demo)

    라이브 시연 — 발표자가 실시간으로 도구를 사용하는 모습을 보여드립니다

    실습 (Hands-on)

    직접 코딩 — 가이드를 따라 각자의 환경에서 직접 구현합니다

    각 섹션은 이 세 가지 형태를 조합하여 진행됩니다

    세션은 세 가지 형태로 진행됩니다. 강의는 슬라이드 중심으로 개념을 설명합니다. 데모는 발표자가 직접 도구를 사용하는 모습을 실시간으로 보여드립니다. 실습은 참가자 여러분이 직접 코드를 작성하는 시간입니다. 대부분의 섹션은 이 세 가지를 적절히 조합하여 진행합니다.

    로드맵: Part 1 — 기초 다지기

    70분 | 강의 중심

    01 AI Foundation — AI Fluency, 4D Framework, Vibe Coding 개념 15분 · 강의
    02 Claude Code Features — CLI 설치, 기본 사용법, 핵심 기능 20분 · 강의+데모
    03 MCP Architecture — MCP 프로토콜, FastMCP로 첫 서버 만들기 25분 · 강의+실습
    전체 로드맵의 Part 1, 기초 다지기입니다. 총 70분으로 강의 중심입니다. 섹션 1에서 AI Fluency와 Vibe Coding의 기초 개념을 다루고, 섹션 2에서 Claude Code를 직접 설치하고 사용법을 익힙니다. 섹션 3에서 MCP라는 도구 연동 프로토콜을 배우며 첫 MCP 서버를 만들어봅니다. Part 1이 끝나면 AI 코딩의 기반 지식이 갖춰집니다.

    로드맵: Part 2 — 실전 구현

    170분 | 실습 중심

    04 Genie MCP — Databricks Genie API, MCP 서버 래핑 30분 · 실습
    05 Skills — 개념에서 실전까지, Custom Skill 만들기 30분 · 강의+실습
    06 Custom Agents — 나만의 에이전트 만들기 25분 · 강의+실습
    07 Hooks — 이벤트 기반 자동화 25분 · 강의+실습
    08 AI Dev Kit — ChatAgent, MLflow Tracing, 배포 파이프라인 15분 · 강의+데모
    09 Build UI — FastAPI 웹 서버, Genie 채팅 UI 구축 30분 · 실습
    10 Deploy — Databricks Apps로 서비스 배포 15분 · 실습
    Part 2는 실전 구현입니다. 총 170분으로 실습 중심입니다. 섹션 4에서 Genie MCP 서버를 만들어 자연어 데이터 조회를 구현하고, 섹션 5에서 Skills 개념을 배우고 직접 Custom Skill을 만들어봅니다. 섹션 6에서 Custom Agents를 만들고, 섹션 7에서 Hooks로 이벤트 기반 자동화를 설정합니다. 섹션 8에서 Databricks AI Dev Kit을 알아보고, 섹션 9에서 웹 UI를 만들어 프론트엔드와 백엔드를 통합합니다. 마지막으로 섹션 10에서 Databricks Apps에 배포합니다. Part 2가 끝나면 실제로 동작하는 AI 웹 앱이 완성됩니다.

    섹션 간 관계를 시각적으로 보여드립니다. Part 1에서는 AI Foundation에서 시작하여 Claude Code, MCP 순서로 기초를 다집니다. MCP 아키텍처 이해가 Part 2의 핵심 기반이 됩니다. Part 2에서는 MCP 지식을 바탕으로 Genie MCP 서버를 만들고, Skills, Agents, Hooks 순서로 Claude Code 확장 체계를 완성합니다. 그 다음 AI Dev Kit과 UI를 거쳐 최종 배포까지 이어집니다. 순서대로 따라가면 자연스러운 흐름이지만, 필요한 섹션만 참고하셔도 됩니다.
