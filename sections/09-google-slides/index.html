    Section 09

    Google Slides

    Vibe Coding으로 슬라이드 자동화

    강의+데모 · 15분

    이번 섹션에서는 Claude Code의 Google Slides 스킬을 활용해서 자연어만으로 프레젠테이션을 자동 생성하는 방법을 알아보겠습니다. 실제로 이 교육 자료 자체도 Vibe Coding으로 만들어졌는데, Google Slides 역시 같은 방식으로 자동화할 수 있습니다.

    학습 목표

    1. Claude Code에서 Google Slides 스킬 활용법 이해
    2. 자연어로 프레젠테이션을 생성하는 워크플로우 학습
    3. Vibe Coding을 활용한 문서 자동화의 가능성 파악
    이번 섹션의 학습 목표는 세 가지입니다. 첫째, Claude Code에서 Google Slides 스킬을 어떻게 사용하는지 이해합니다. 둘째, 자연어 명령만으로 프레젠테이션을 생성하는 전체 워크플로우를 학습합니다. 셋째, 슬라이드를 넘어서 다양한 문서 자동화의 가능성을 살펴봅니다.

    왜 슬라이드를 코드로 만드는가?

    기존 방식의 문제

    슬라이드 디자인에 시간 소모 • 반복적인 레이아웃 작업 • 일관성 유지 어려움

    Vibe Coding 방식

    자연어로 내용 전달 • AI가 디자인/구조화 • 수정도 대화로 해결

    결과

    콘텐츠에 집중 • 제작 시간 단축 • 템플릿 기반 일관성 확보

    왜 슬라이드를 자동화해야 할까요? 기존에는 슬라이드를 만들 때 디자인에 많은 시간을 쓰고, 반복적인 레이아웃 작업을 하고, 여러 사람이 만들면 일관성이 깨지기 쉽습니다. Vibe Coding 방식에서는 자연어로 내용만 전달하면 AI가 구조화하고 디자인합니다. 수정이 필요하면 대화로 요청하면 됩니다. 결과적으로 콘텐츠에 집중할 수 있고, 제작 시간이 크게 단축됩니다.

    Google Slides 스킬

    Claude Code의 /google-slides 스킬이 제공하는 기능

    프레젠테이션 생성

    새 Google Slides 프레젠테이션을 자동으로 생성

    템플릿 & 서식

    Databricks 템플릿 적용, 표, 차트, 이미지 삽입

    콘텐츠 관리

    슬라이드 추가/수정/삭제, 텍스트 서식, 레이아웃 조정

    Google Workspace 연동

    gcloud CLI + Google Slides API를 통한 직접 제어

    Google Slides 스킬은 Claude Code에서 slash 명령어로 호출할 수 있습니다. 프레젠테이션 생성, 템플릿 적용, 표와 차트 삽입, 슬라이드 추가/수정/삭제 등 다양한 기능을 제공합니다. 내부적으로는 gcloud CLI와 Google Slides API를 사용해서 Google Workspace와 직접 연동됩니다.

    데모 시나리오

    자연어 명령 한 줄로 프레젠테이션을 만들어봅니다

    프롬프트 작성 자연어로
    내용 전달
    →
    스킬 실행 Google Slides
    API 호출
    →
    결과 확인 Google Slides
    에서 열기     
    # Claude Code에서 이렇게 요청합니다
> "Databricks Unity Catalog 소개 프레젠테이션을 5장으로 만들어줘.
   각 슬라이드에는 제목, 핵심 내용, 스피커 노트를 포함해줘."
    데모 시나리오를 설명드리겠습니다. 워크플로우는 간단합니다. 자연어로 프롬프트를 작성하면, Claude Code가 Google Slides 스킬을 실행해서 API를 호출하고, 완성된 프레젠테이션을 Google Slides에서 바로 확인할 수 있습니다. 예를 들어, "Unity Catalog 소개 프레젠테이션을 5장으로 만들어줘"라고 요청하면 됩니다.

    데모: 슬라이드 생성 LIVE

    Claude Code에서 자연어로 프레젠테이션을 생성합니다 

    # Step 1: Google 인증 확인
$ gcloud auth list
# 활성 계정이 표시되는지 확인
    # Step 2: Claude Code에서 스킬 호출
> /google-slides

# "Databricks Unity Catalog 소개 프레젠테이션을 만들어줘.
#  5장 구성으로, 각 슬라이드에 제목과 핵심 내용을 넣어줘.
#  Databricks 템플릿을 적용해줘."
    # Step 3: 생성된 프레젠테이션 URL 확인
# https://docs.google.com/presentation/d/xxxxx/edit
# 브라우저에서 바로 열어서 확인!
    실제 데모를 진행합니다. 먼저 gcloud auth로 Google 인증이 되어있는지 확인합니다. 그 다음 Claude Code에서 /google-slides 스킬을 호출하고 자연어로 프레젠테이션 내용을 설명합니다. Claude Code가 Google Slides API를 호출해서 프레젠테이션을 생성하면 URL이 반환되고, 브라우저에서 바로 확인할 수 있습니다. [실제로 실행하여 결과를 보여주세요]

    데모 결과 리뷰

    자동 생성된 프레젠테이션의 구조를 살펴봅니다

    자동으로 생성된 것

    슬라이드 레이아웃 • 제목/본문 텍스트 • 서식 & 색상 • 스피커 노트

    추가 수정도 자연어로

    "3번째 슬라이드에 아키텍처 다이어그램 설명을 추가해줘"

    핵심 포인트

    콘텐츠만 전달하면 AI가 구조화 → 검토 → 수정 요청의 반복

    생성된 결과를 리뷰해봅시다. 슬라이드 레이아웃, 제목과 본문, 서식과 색상, 스피커 노트까지 모두 자동으로 생성되었습니다. 수정이 필요하면 역시 자연어로 요청하면 됩니다. 예를 들어 "3번째 슬라이드에 내용을 추가해줘"라고 하면 됩니다. 핵심은 콘텐츠에만 집중하고, 나머지는 AI에게 맡기는 것입니다.

    활용 팁 & 확장 아이디어

    효과적인 프롬프트

    • 슬라이드 수를 명시
    • 각 슬라이드 주제 나열
    • 청중과 목적 설명
    • 원하는 톤/스타일 지정

    확장 활용

    • 주간 리포트 자동 생성
    • 고객 미팅 자료 준비
    • 데이터 분석 결과 발표
    • Google Sheets/Docs 연동
    Section 05에서 배운 Skills의 실전 활용 사례입니다
    활용 팁과 확장 아이디어를 공유합니다. 효과적인 프롬프트를 작성하려면 슬라이드 수를 명시하고, 각 슬라이드의 주제를 나열하고, 청중과 목적을 설명하는 것이 좋습니다. 확장 활용으로는 주간 리포트 자동 생성, 고객 미팅 자료 준비, 데이터 분석 결과를 슬라이드로 변환하는 것 등이 있습니다. Google Sheets나 Docs 스킬과 결합하면 더 강력한 자동화가 가능합니다. 이것이 바로 Section 05에서 배운 Skills의 실전 활용 사례입니다.

    요약

    • Google Slides 스킬로 자연어 → 프레젠테이션 자동 생성
    • gcloud CLI + Google Slides API 기반의 직접 제어
    • 콘텐츠에 집중하고 디자인은 AI에게 → 생산성 향상

    Vibe Coding은 코드만이 아닙니다.
    문서, 슬라이드, 스프레드시트 — 모든 것을 자동화할 수 있습니다.
